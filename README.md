#  Screen Recorder Web App

A full-stack screen recording web application that allows users to record their screen, preview recordings, upload videos, and download it directly into device.

---

##  Features

-  Screen recording using MediaRecorder API
-  Live preview before upload
-  Upload recordings to backend
-  Store videos on server
-  Download recorded videos
-  Upload progress indicator
-  Scalable backend architecture

---

##  Tech Stack

### Frontend
- React (TypeScript)
- Vite
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

## ⚙️ How It Works

1. User starts screen recording
2. Video is captured using MediaRecorder API
3. Recording is previewed in browser
4. User uploads video to backend
5. Backend stores video in server
6. Video is accessible via URL
